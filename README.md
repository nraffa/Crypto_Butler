# Crypto_Butler
A telgram bot that informs the price, hour change(%) and day change(%) of the following cryptocurrencies:
-BTC
-ETH
-ETC
-ADA
-DOGE
-XRP

## How can you find the bot?
Go to the following link: https://t.me/CryptoMayordomo_bot
